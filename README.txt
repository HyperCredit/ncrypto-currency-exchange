This is a simple library for interacting in a generic manner with a number
of cryptocurrency exchanges.

Please ensure you've read and understand the disclaimer in the license above.
This software comes with absolutely no warranty, and anyone wishing to use it
should review the code themselves to ensure they're happy it works in as
they expect. This software is not intended for non-expert use, and you should
be comfortable with cryptocurrency design and usage, the nature of the exchanges
being traded on, and currency trading.


To get started, please look at the example provided under the "Examples"
folder, of how to construct an exchange API class, and make calls to it.
